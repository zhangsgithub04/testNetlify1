Version 4 Grammar of a tinny language (expression, term and factor, again without exponentation operator, to avoid left recursion in version 3. Exponentation will be supported via built-in function later !!!

  1. STATEMENT-> ASSIGNMENTSTATEMENT
  2. ASSIGNMENTSTATEMENT-> id = EXPR ;
  3. EXPR -> TERM {(+|-) TERM}
  4. TERM -> FACTOR {(*|/) FACTOR}
  5. factor -> '(' expr ')' | identifier | literal constant
